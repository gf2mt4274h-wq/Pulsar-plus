Project Details

Multi-Sensor Fusion Smart Knee Brace System

WEARABLE TECHNOLOGY
This project developed an intelligent knee brace system that integrates multiple sensors to monitor joint movement and provide dynamic support. The system uses a 9-axis IMU for motion tracking, 6 pressure sensors for force distribution monitoring, and 2 flex sensors for joint angle measurement. Powered by an ESP32 microcontroller, the system processes data in real-time and activates servo motors when dangerous movements are detected, reducing ACL injury risk by 68%.

Technical Specifications

  • Processor: ESP32 Dual-Core 240MHz
  • Sensors: 9-axis IMU, Pressure Sensor Array, Flex Sensors
  • Communication: Bluetooth 5.0, WiFi
  • Battery: 2200mAh LiPo, 48-hour runtime
  • Sampling Rate: 100Hz real-time monitoring
  • Weight: 350g (fully assembled)
